  1. class SVDQueryEngine extends AnyRef

    This class implements useful methods to query and discover an SVD model.

    This class implements useful methods to query and discover an SVD model. It is inspired by the book Advanced Spark Programing v2, chapter 6, but has been modified to work with org.apache.spark.mllib.linalg.distributed.IndexedRowMatrix instead of org.apache.spark.mllib.linalg.distributed.RowMatrix. This way, the document id is embedded in the matrix instead of relying on a "zipWithUniqueId trick" that works only when we do everything (preprocessing, modeling, querying) inside the same spark session.
